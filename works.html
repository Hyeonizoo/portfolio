이 프로젝트를 진행하며-

기획, 디자인, 프로그래밍 전공이 모여 처음으로 진행한 팀 프로젝트입니다. 처음 협업을 해보며 제작 프로세스의 전반적인 이해와 더불어 많은 것을 배울 수 있었습니다. 혼자 작업할 때와는 달리 서로를 의지할 수 있다는 점에도 큰 매력을 느꼈습니다. 또, 팀원들과 같은 주제와 목표를 가지고 커뮤니케이션을 하는 과정과, 여러 의견을 조율해가며 작업물을 발전시키는 작업이 즐거웠습니다. 이런 과정을 거쳤기 때문에 공모전에서 특별상이라는 좋은 결과를 얻을 수 있었던 것 같습니다.

더 나은 코드를 짜기 위한 노력.

‘어금니’ 앱에서 제공하는 테스트 기능을 홍보 웹사이트에서도 체험해볼 수 있도록 바닐라 자바스크립트를 사용해 ‘성인 ADHD 유형 테스트’ 페이지를 제작했습니다.

테스트 결과값을 미리 구성해 그에 맞게 라디오 버튼에 value 값을 주었습니다. active 클래스를 활용해 체크해야 할 항목을 강조했습니다.

체크한 각 문항 value값을 배열에 담아 모두 더해 계산한 후, if문을 활용해 각 결과값에 맞는 페이지로 이동하도록 했습니다. 올바른 결과를 도출하기 위해 모든 항목에 체크해야 결과 페이지가 나오고, 체크할 항목이 남은 경우에는 alert 창을 띄워 모든 항목에 체크하도록 유도했습니다.

setInterval을 사용해 일정 시간마다 그래프와 숫자가 증가하는 인터랙션을 제작하였습니다.

gsap을 사용해 요소들이 둥둥 떠다니는 인터랙션을 제작하였습니다.

  • #지속적인

이 프로젝트를 진행하며-

졸업전시회 작품으로 제작한 4050세대를 위한 퇴직 후유증 케어 서비스 '이담'을 소개하는 웹사이트입니다. 해당 프로젝트에서 웹 제작을 맡아 진행하였으며, 팀원들과 함께 기획 전반과 디자인 파트에도 참여하였습니다. '퇴직한 4050세대'라는 특수한 조건의 사용자층을 고려해, 사용자의 이름이 적힌 근로 계약 해지 통보서를 제공함으로써 보다 폭넓은 사용자층이 상황에 더 몰입하고 흥미를 느낄 수 있도록 유도했습니다.

더 나은 코드를 짜기 위한 노력.

jQuery의 addClass를 활용해 근로 계약 해지 통보서를 띄우고 닫을 수 있습니다. 부서와 직급을 배열에 담아 사용했으며, 랜덤함수 Math.random()을 사용해 클릭시마다 다른 값이 호출됩니다. 미리 사용자 이름을 받아와 통보서에 호출하여, 통보서를 열었을 때 사용자의 이름이 적혀있도록 했습니다.

addClass를 활용해 tab 기능이 담긴 정보구조도를 제작했습니다. active되지 않은 기본 list는 첫 번째 배열(tabArray)인 ‘1단계, 2단계, 3단계’처럼 간략하게 제공하고 클릭하여 active된 list는 두 번째 배열(selectArray)의 ‘1단계 | 퇴직자 후유증 심각단계’처럼 추가 정보와 함께 제공해 가독성을 높였습니다.

setInterval을 사용해 단계별 화면을 제공하였습니다. 반복문을 사용해 같은 단계에 해당하는 각각의 요소가 동시에 active될 수 있도록 했습니다. 해당 과정을 겪으며 이미지를 바꾸는 것에 어려움이 있었는데, js에서의 이미지를 부를 때 이미지 경로 설정에 유의해야 함을 다시 한 번 느꼈습니다.

웹사이트 메인에 css 애니메이션을 적용헤 서비스 슬로건을 보여주었습니다. 슬로건 애니메이션이 끝나면 기본 메인화면이 보여지도록 했습니다.

처음에 javascript를 사용해 스크롤 애니메이션으로 적용하려고 했었는데, 꼭 봐야하는 정보는 아니기 때문에 불필요한 기능이라고 판단해 reload시 한 번만 제공되도록 하였습니다.

이 프로젝트를 진행하며-

프로그래밍을 처음 배우며 제작했던 '교보문고'를 주제로 한 웹사이트를 발전시켰습니다. html과 css만으로 제작했던 기존 작업물에서, 다시 탄탄하게 마크업을 작성하고 '체험형 콘텐츠'와 결합하여 사용자가 웹사이트와 상호작용하며 재미를 느낄 수 있도록 재구성하였습니다. 같은 주제로 웹사이트를 다시 제작해보며, 1년 사이에 많이 성장했음을 느낄 수 있었던 의미있는 시간이었습니다.

더 나은 코드를 짜기 위한 노력.

jQuery draggable 기능을 사용해 요소를 드래그해 움직이게 할 수 있도록 했습니다. target에 적용시켜 특정 요소를 움직이는데, 이때 containment 속성을 사용해 부모요소 내에서만 움직일 수 있도록 했습니다.

같은 기능을 JavaScript를 사용해 제작해보았습니다. 화면 전체를 감싸는 wrap(.screen)에 적용시켰습니다.

해당 부분의 기본 커서는 grab을 사용했으며, ‘.screen_loading’ 클래스를 사용해 요소를 클릭하거나 클릭하고 있을 때 cursor: grabbing 효과를 주어 시각적인 디테일을 주었습니다.

이때 요소의 scale을 약간 키워, 오브젝트와 상호작용하고 있음을 부각했습니다.

drag & drop 기능이 전체 screen과 target에 모두 들어가기 때문에, target 클릭 시 이벤트 버블링이 일어나는 문제가 있었습니다.
event.stopPropagation()을 사용해 버블링 문제를 해결하였습니다.

addClass를 사용해 tab 기능을 구현하였습니다. gnb 클릭시 해당 li에 active 클래스가 적용되며, 그에 해당하는 콘텐츠를 띄우도록 했습니다.

랜덤 함수(Math.random()을 사용해 현재 보고 있는 사람을 임의적으로 나타냈습니다.

setInterval을 사용해 실시간 hot 순위와 그에 따른 콘텐츠가 순차적으로 반복해서 보이도록 했습니다. 현재 보고 있는 사람의 경우 임의적으로 나타낸 것이기는 하지만 setInterval 함수 내에 다시 작성하여 reload시가 아닌 setInterval될 때마다 값이 바뀌도록 했습니다.
